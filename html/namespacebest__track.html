Detailed Description

Concatenates storm tracks from w2segmotionll.

This package is approximately equivalent to w2besttrack with the potential for additional features and greater flexibility. This python version was converted from Ryan Lagerquist's MATLAB code ryan_best_tracks.m and associated files.

Author
David Harrison
Date
July 2016
